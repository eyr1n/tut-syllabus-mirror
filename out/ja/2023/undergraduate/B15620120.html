授業の目標
日本語
コンピュータを用いた構造解析の基礎となる数値解析手法を学習すると共に，建設構造物のマトリクス構造解析法による数値解析手法の基礎を学習する．また，トラス構造と簡単なラーメン構造のFORTRAN言語を利用した構造解析プログラムの実際についても講義する．
英　語
Learn technologies of the numerical analysis by using finite element method (FEM) analysis. The base of the numerical analysis technique by the FEM analysis method of a construction structure is learned. Moreover, the structural analysis programs using the FORTRAN language for plain truss structures and plain frame structures are developed.
 
授業の内容
日本語
第1部：有限要素法による構造解析（中澤担当）
有限要素法による構造解析の基礎となる「マトリクス演算」，「連立1次方程式の解法」などのプログラミングをFortran言語を用いて学習する．また、平面トラス、平面ラーメン構造を対象とし、有限要素法プログラムを用いた構造解析法を学習する．
第　１週　ガイダンス、環境設定
第　２週　建設分野における構造解析の利用について（外部講師による特別講演を予定）
第　３週　プログラミングの基礎１
第　４週　プログラミングの基礎２
第　５週　行列の演算（行列の積、連立方程式の解法）
第　６週　平面トラス構造の応力解析１
第　７週　平面トラス構造の応力解析２
第　８週　平面ラーメン構造の応力解析１

第２部：トラスおよびラーメン構造の構造解析（瀧内、中澤担当）
トラス構造およびラーメン構造の基礎理論を学び、構造解析を行いながら、応力解析の方法について学習する。3DCADのRhinoceros上でこの授業で作成したプログラムを実装し解析に利用する。
第　９週　後半ガイダンスと環境構築
第１０週　Pythonを用いたプログラミング基礎
第１１週　Rhinoceros,Grasshopperを用いたプログラミング基礎
第１２週　マトリクス法の基礎理論と構造解析プログラムの使用法
第１３週　FEM解析を用いたパラメトリックデザイン その１
第１４週　FEM解析を用いたパラメトリックデザイン その２
第１５週　FEM解析を用いたパラメトリックデザイン その３
