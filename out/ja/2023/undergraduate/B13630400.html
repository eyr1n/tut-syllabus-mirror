授業の目標
日本語
情報系、コンピュータ科学の分野の学問の基盤となる計算機の構成と設計に関する中心的な考え方を取得し、ハードウェアとソフトウェアの間の関係性を理解すること。
英　語
The goal is to obtain the central concepts in computer organization and design, and to understand the relationship between hardware and software. Especially, these concepts and ways of thinking are very important foundation for Computer Science and related information processing technique.
 
授業の内容
日本語
本講義では、コンピュータの基礎的な動作原理から、近年のコンピュータ技術を理解する上でポイントとなる発展的な原理までを網羅的にカバーします。初学者にもわかりやすく説明するように工夫をした内容に構成しております。

1週目 イントロダクション
2週目 計算機の性能
3週目 命令セットとメモリアドレス
4週目 マシン語と関数呼び出し
5週目 再帰とメモリ管理とデータ表現法
6週目 命令セットの比較と算術法
7週目 データ表現法と浮動小数点
8週目 プロセッサの設計の基礎
9週目 パイプラインによる高速化
10週目 パイプライン制御と分岐予測
11週目 メモリ階層の利用とキャッシュメモリ
12週目 キャッシュ最適化技術とソフトウェア性能
13週目 命令レベル並列性、マルチスレッド、SIMD
14週目 省電力技術、ディペンダブル技術
15週目 GPUからクラウドまで、講義のレビュー
16週目 定期試験

英　語
In this course, we comprehensively cover the topics from foundamental principles of computer science to the advanced principles for understanding modern computer organization.

1st week: Introduction to computer architecture
2nd week： Performance of computer
3rd week: Instruction set and memory address
4th week: Machine code and Procedure calls
5th week: Recursions, memory management, and data representations
6th week: Comparison of ISAs and arithmetic
7th week: Data representation and floating point numbers
8th week: Introduction for processor design
9th week: Pipeline for performance
10th week: Pipeline control and branch prediction
11th week: Memory hierarchy and caches
12th week: Cache optimization technique and software performance
13th week: Instruction level parallelism, multithread, SIMD
14th week: Low-power technique and dependable computing
15th week: From GPU to cloud, Review
15th week: Final exam
