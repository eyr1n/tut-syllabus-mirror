Objectives of class
Japanese
データサイエンスによく用いられる言語としてのPython言語の基礎と代表的な応用事例を修得する． NumPyやSciPyなどの代表的なパッケージを用いて，機械学習とデータサイエンスの初歩を理解する．
English
This course provides an introduction to the Python language as a language commonly used in data science and its typical applications. Specifically, this course covers the basics of machine learning and data science using typical packages such as NumPy and SciPy.
