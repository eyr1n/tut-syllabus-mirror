Contents of class
Japanese
計算機を用いた問題解決の対象となるシステムの多くは、グラフ・ネットワークなどの離散構造をしている。そこで、離散構造を持つシステムをモデル化し、分析するツールとして、グラフ・ネットワーク理論を取り上げる。

１週目―３週目　数学的準備と離散数学の基礎概念
・第1週（回）集合， 写像， 関数
・第2週（回）論理，数学的帰納法， 背理法
・第3週（回）鳩の巣原理(ディリクレの原理)

４週目―７週目　グラフの基礎概念
・第4週（回）路，閉路，連結性
・第5週（回）木、カットセット
・第6週（回）オイラ－路とハミルトン路
・第7週（回）．．．　小テスト

８週目―９週目　平面グラフと双対グラフ
・第8週（回）平面グラフ
・第9週（回）地図の彩色

１０週目―１２週目　ネットワーク設計とグラフの連結性
・第10週（回）ネットワークと最適化
・第11週（回）2部グラフとマッチング
・第12週（回）トポロジカルソート

１３週目―１６週目　最短路問題入門、総括
・第13週（回）動的計画法と最短路問題
・第14週（回）圏論入門　モノの集まり　から　森羅万象の関係へ
・第15週（回）総括
・第16週（回）テスト

English
Many systems that become the subject of problem solving using a computer system, have a discrete structure, such as a graph network. Therefore, we model the system with a discrete structure, and
learn the graph network theory as analysis tools.

1-3 weeks mathematical preparation and discrete mathematics of the basic concepts
・1st week: set, mapping, function
・2nd week: mathematical induction, reductio ad absurdum
・3rd week: Pigeonhole principle (Dirichlet's box principle)

4-7 weeks basic concepts of graph theory
・4th week: path, cycle, connectivity
・5th week: tree, cut set
・6th week: Euler path, Hamilton path
・7th week: mid-term exam.

8-9 weeks planar graph and dual graph
・8th week:
・9th week:

10-13 weeks network design and connectivity of graph
・10th week: networks and optimization
・11th week: bipartite graph and matching
・12th week: topological sort

13 -16 weeks introductory shortest path problem
・13th week: Dynamic programming and shortest path problem
・14th week: Introduction to Category Theory (From set of things To relation of Everything)
・15th week: Review
・16th week: Exam.

Japanese

2) To master algorithms to solve the minimum spanning tree problem and shortest path problem.
