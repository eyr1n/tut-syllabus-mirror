Contents of class
Japanese
第1回　機械要素の設計（標準規格，安全係数，寸法公差とはめあい）
第2回　締結法（ねじ，ねじ部品，キー，スプライン，ピン）
第3回　軸と軸継手（軸の強度，軸のたわみ，永久継手，クラッチ）
第4回　軸受（転がり軸受，すべり軸受）
第5回　歯車（歯車の種類，歯車の設計）
第6回　ばね（コイルばね，板ばね，皿ばね，トーションバー）
第7回　マイクロ機械要素
第8回　まとめ45分，期末試験45分

本学の新型コロナウィルス感染拡大防止のための活動基準の変更に伴い、授業内容および成績の評価法に変更が生じる場合があります。

English
1st week; Introduction of Machine Elements (Standards, Factor of Safety, Tolerance and Fit)
2nd week; Transmission elements (Screws, Keys, Splines, and Pins)
3rd week; Shaft and Shaft couplings (Permanent coupling and Clutch)
4th week; Bearings (Rolling bearing and Sliding bearing)
5th week; Gears (Types of gears and Designing of a gear pair)
6th week; Springs (Helical spring, Flat spring, Disc spring, and Torsion bar)
7th week; Micro Machine Elements
8th week; Summary: 45 min. and Regular examination: 45 min

Goals to be achieved
Japanese
次項について理解すること,
1) 機械設計の概略
2) 機械要素の名前，機能，性能
3) 機械要素の基本的な構造と製造法

English
Understand following items,
1) outline of machine design
2) name, function, performance of each machine element
3) fundamentals, structure manufacturing process of each machine element
4) design parameters of each machine element
