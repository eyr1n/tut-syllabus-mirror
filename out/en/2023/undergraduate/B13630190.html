Objectives of class
Japanese
フィードバック制御系の基礎事項を学習した後に、システムの時間特性、制御系の設計法を習得し、さらに現代制御理論の基礎的な事項を学ぶ。
English
This course provides the basis of feedback control system, stability and time characteristic of system, and design method for the control system. It also covers modern control theory.
 
Contents of class
Japanese
企業においてロボット制御の研究開発業務に携わった経験のある教員が、制御工学の基礎から実際の制御系の設計まで講義を行う。

1週目　制御工学とは
2週目　ブロック線図、フィードバック制御系の基礎
3週目　ラプラス変換と周波数領域でのシステム表現
4週目　伝達関数
5週目　時間応答と周波数応答（ナイキスト線図とボード線図）、基本要素の時間応答と周波数応答
6週目　２次要素の時間応答と周波数応答
7週目　ラウス・フルヴィッツの安定判別法
8週目　ナイキストの安定判別法
9週目　フィードバック制御の特性（過渡特性と定常特性）
10, 11週目　周波数領域での制御系の設計, PID制御の制御系の設計
12週目　MATLABを用いた制御系の設計
13週目　システムの状態空間表現
14,15週目　状態変数表現を用いた制御系設計
16週目　期末試験

本学の新型コロナウィルス感染拡大防止のための活動基準の変更に伴い、授業内容および成績の評価法に変更が生じる場合があります。
授業実施形態が変更になる場合は，GoogleClassroomまたは教務情報システムより通知します。
English
Teachers with experience in research & development on robot control in a company will speak about knowledge from the basics of control engineering to the design of actual control systems.

1st week　Introduction of control theory
2nd week　Block diagram　and basis of feedback control system
3rd week Laplace transformation and system representation in the frequency domain
4th week　Transfer function　
5th week　Time response and frequency response (Nyquist diagram and Bode plot) of basic elements
6th week　Time response and frequency response of Second order element
7th week　Stability (Routh-Hurwitz criterion)
8th week　Stability (Nyquist stability criterion)
9th week　Steady state and transient responses
10th and 11th week　Design of feedback control systems in frequency domain, Design of PID control system
12th week Design of feedback control system using MATLAB
13th week　State variable representation of systems
14th and 15th week　Design of feedback control systems in state variable representation of systems
16th week　Final Exam

Goals to be achieved
Japanese
（１）制御システムの基礎
　１） システムの表現法（伝達関数とブロック線図）を理解できる
　２）周波数応答の表示法としてのナイキスト線図、ボード線図を理解できる
　３）ラウス・フルビッツとナイキスト安定判別法によってシステムの安定判別ができる
（２）システムの時間特性
　１）過渡特性と定常特性を理解できる
　２）速応性を理解できる
　３）定常偏差を理解できる
（３）周波数領域でのフィードバック制御系の設計
　１）周波数応答によるシステムを設計できる
（４） 状態方程式と状態フィードバック
　１）制御系を表現する状態方程式を書きくだすことができる。
　２）与えられた制御系に対して，その可制御性と可観測性を調べることができる。
　３）極配置制御による安定な状態フィードバック系の係数ベクトルを計算できる。
　４）状態観測器を用いた状態フィードバック系を構成することができる。
English
（１）Basic concept of control system
　1) Understand representation of the system using transfer function and block diagram
　2) Understand Nyquist diagram and Bode diagram as a representation of frequency responses
　3) Be able to use Routh and Hurwitz stability criterion and Nuquist stability test to test the stability of a system
（２）Time response of system
　1) Understand transient and steady state characteristics.
　2) Understand responsiveness of systems
　3) Understand steady-state error
（３）Design of feedback system in frequency domain
　1 ) Be able to design feedback system in frequency domain
（４）State equation and state-feedback
　1) Be able to describe state equation representing systems
　2) Be able to check controllability and Observability of system
　3) Be able to design state-feedback system using pole assignment method
　4) Be able to design state-feedback system using state observer
