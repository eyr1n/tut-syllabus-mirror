Goals to be achieved
Japanese
各実験テーマの達成目標は以下のとおりである。

* マイクロプロセッサ
教育用ボードKUE-CHIP2を用いて計算機の動作原理を理解する。アセンブリ言語プログラムを機械語に変換し、命令の取り出し・解読・実行のサイクルを観察することで計算機のソフト・ハード両面の理解を実践的に深める。

* CPUの論理設計
ハードウェア記述言語VerilogとFPGAシミュレーションツールを利用し、プロセッサの設計および論理合成方法を学習する。CPU論理設計の基本的な手法を理解することが本実験の目的である。

* ネットワークセキュリティ
DoS攻撃とは、ネットワーク上のサーバに攻撃を行い、正常なサービスの提供を不能にする攻撃のことである。サーバへのDoS攻撃を検出し、フィルタリングすることによりDoS攻撃への対処方法を習得する。

* データ情報学
本実験では、プログラミングを通じて、音・画像などの数値データを分析・解析・評価することで、基本的なデータ処理を習得することを目的とする。各自が今後研究室で取り組むであろう高度な情報処理の基礎となることが期待できる。

* ヒューマン・ブレイン情報学
本実験では、ヒトを対象とする心理物理実験の実験実施・参加者体験を通して、実践的なヒト感覚・知覚測定法、プログラミング技術、および報告書の作成方法を習得する。

* メディア・ロボット情報学
本実験では、小型移動ロボットを題材としてロボットプログラミングの基礎技術を実践的に学習する。具体的には、Robot Operating System (ROS) ミドルウェアとPythonを用いて、各種センサのデータ処理とモータ制御を行う。また限られた時間・機材のもとで、チームで協力して課題を設定し実装を計画的に進める能力を養う。

* 計算機数理科学
本実験では、直観に反する確率パズルを題材に、C/C++を用いたモンテカルロ法に基づく実験を行う。実験で得られたデータからパズルの解の推定および考察を行うことで、プログラム技術および論理的思考力を発展させることを目的とする。
English
The purposes for each themes are as follows.

* Microprocessor
You understand the principal of computer by using an educational board (KUE-CHIP2).
You understand the hardware and software of computer through translating assembly code into machine code and observing cycles for the load, decode, and implementation of commands.

* Logical Design of CPU
You learn how to design and synthesize a processor using Verilog HDL and FPGA simulation tools. The goal is to understand the basic techniques of CPU logical design.

* Network Security
DoS attack is an attack to network servers and to destruct normal service. You learn how to cope with the DoS attack by detection and filtering the attack.

* Data informatics
The aim of this theme is to learn fundamental data processing by analyzing and evaluating numerical data, such as sounds and images, through programming language coding.

* Human Brain Informatics
In this experiment, you learn practical human sensory and perceptual measurement methods, programming techniques, and report writing methods through conducting and participating in psychophysical experiments.

* Media Informatics and Robotics
You learn basic robot programming techniques using a small mobile robot. Specifically, by using Robot Operating System (ROS) middleware and Python, you process sensor data and control the robot's motors. You also develop the ability to work as a team to set goals and implement them under limited time and equipment.

* You study the Monte Carlo simulation using C/C++ on subjects of counter-intuitive probability problems. The purpose is to develop programming and logical thinking skills by estimating and discussing the solutions from the data obtained in the simulations.
