Objectives of class
Japanese
C言語の拡張として設計されたオブジェクト指向言語であるC++言語でのプログラミングを修得する。
企業の研究所で研究開発に携わっていた教員が、その経験を生かして、C++言語でのプログラミングについて講義を行う。
English
This class teaches programming skill of object-oriented language using C++ language that is designed by extending C language.
